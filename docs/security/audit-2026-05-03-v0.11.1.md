> **No security delta from v0.11.0.**
> v0.11.1 ships two changes, both wp.org-compliance-only :
>   - Chart.js (admin-only timeline charts) is now bundled locally under `assets/js/vendor/chart.umd.min.js` instead of loaded from `cdn.jsdelivr.net`. This actually *improves* the security posture by removing a runtime dependency on an external host, but is otherwise functionally identical.
>   - A new `plugin-check` job is added to `.github/workflows/ci.yml` running the official WordPress `plugin-check-action`. Same automated checks as wp.org reviewers — catches future regressions before submission.
>
> No new input surface, no DB writes, no permission changes. All 9 surfaces from the v0.9.3 / v0.10.0 / v0.11.0 reports remain valid. PHPCS still 0 findings.
