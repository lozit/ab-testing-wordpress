> **No security delta from v0.10.1.**
> v0.11.0 ships one feature: a per-URL `noindex` toggle. The new `Abtest\UrlSettings` class is a thin wrapper over a single WP option (`abtest_url_settings`), only invoked from the admin save handler (already gated by `current_user_can('manage_options')` + nonce) and from two read-only frontend hooks (`send_headers` + `wp_robots` filter) that emit standard SEO directives. No new input surface, no DB writes outside of `update_option()`, no outbound HTTP, no new permission paths.
>
> All 9 surfaces from the v0.9.3 / v0.10.0 reports remain valid. `composer audit` clean. PHPCS 0 findings. 97 unit + 15 integration tests pass.
